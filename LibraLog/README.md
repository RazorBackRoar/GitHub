A macOS library management application built with Python and PySide6.

## Features

- Personal library catalog management
- Book tracking and organization
- Reading progress monitoring
- Dark theme UI

## Requirements

- macOS (Apple Silicon)
- Python 3.10+
- PySide6

## Installation

```bash
pip install -e .
```

## Development

```bash
# Run the app
python -m libralog

# Build DMG
cd ../razorcore && ./universal-build.sh LibraLog
```
